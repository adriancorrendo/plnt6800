1.1 Main Differences

The key divergence lies in their treatment of TRUTH.

Frequentism assumes the existence of a fixed, true value. Parameters are fixed, and randomness comes from data variation. It‚Äôs called ‚Äúfrequentism‚Äù because it relies on the frequency of events under repeated sampling.

üé≤ Example: To estimate the probability of rolling a 6, Frequentists say: ‚ÄúIf we roll a die infinitely many times, the proportion of 6s will approach 1/6.‚Äù

Inference is based on idealized, repeated experiments ‚Äî even ones we‚Äôve never performed.

In contrast, Bayesianism does not assume a fixed truth. It centers on:

PROBABILITIES: All knowledge is probabilistic ‚Äî not true/false, but likely/unlikely.

BELIEFS: Prior knowledge is incorporated into analysis via prior distributions. Even when we know nothing (üëâ uninformative priors!), Bayesianism lets us model uncertainty.

üé≤ Bayesian Dice: ‚ÄúWe are a priori 16.7% certain we‚Äôll get a 6.‚Äù

Bayesian inference updates beliefs based on observed data:

\[ \text{Posterior} = \frac{\text{Likelihood} \times \text{Prior}}{\text{Evidence}} \]

And to compare competing models or beliefs, we use the Bayes Factor:

\[ \text{Bayes Factor} = \frac{\text{Posterior}}{\text{Prior}} = \frac{0.334}{0.167} = 2 \]

This means the updated belief is twice as likely, given the data.

1.1.1 Summary:

  • Frequentism: Models are fixed; randomness is in the data.
  • Bayesianism: After being observed, the data is considered fixed; uncertainty lies in the model (parameters and predictions).
Tip

For many simple models, both approaches yield similar conclusions.

üí° Hint: Consider uninformative priors!

Note

Bayesian logic mirrors human reasoning:

  1. We collect data.
  2. We have prior beliefs.
  3. We combine both to update our beliefs.

1.2 Bayes Theorem

\[ P(A_{\text{true}} \mid B) = \frac{P(B \mid A_{\text{true}}) \cdot P(A_{\text{true}})}{P(B)} \]

\[ \text{Posterior} = \frac{\text{Likelihood} \times \text{Prior}}{\text{Evidence}} \]

1.3 The Priors

Priors formalize our beliefs as probability distributions ‚Äî based on:

  1. The nature of the variable (discrete/continuous)
  2. Our prior knowledge or lack thereof
Tip

üß† Bayesian Perspective on Uncertainty

In Bayesian analysis, uncertainty is placed on:

  • Parameters (they are random variables with prior/posterior distributions)
  • Predictions (future outcomes are modeled probabilistically)

Once data is observed, it‚Äôs considered fixed.

So rather than saying ‚Äúdata is fixed, model is random,‚Äù it‚Äôs more precise to say:

‚ÄúBayesian analysis treats unknown parameters and future data as random variables, while the observed data is fixed. The uncertainty lies in our beliefs about parameters and predictions, not in the data we‚Äôve already seen.‚Äù

1.4 üìà Visualization of Bayesian Updating

x <- seq(0, 1, length.out = 300)
prior <- exp(-((x - 0.3)^2) / 0.05)
likelihood <- exp(-((x - 0.7)^2) / 0.02)
posterior <- exp(-((x - 0.6)^2) / 0.01)

# Normalize
prior <- prior / sum(prior)
likelihood <- likelihood / sum(likelihood)
posterior <- posterior / sum(posterior)

df <- data.frame(x = x, Prior = prior, Likelihood = likelihood, Posterior = posterior)

df_long <- df %>% pivot_longer(cols = -x, names_to = "Distribution", values_to = "Density")

ggplot(df_long, aes(x = x, y = Density, color = Distribution)) +
  geom_line(size = 1.2) +
  labs(title = "Bayesian Updating: Prior √ó Likelihood ‚Üí Posterior",
       x = "Parameter (\u03B8)", y = "Density") +
  theme_classic() +
  scale_color_manual(values = c("tomato", "steelblue", "darkgreen"))

1.5 Credible vs. Confidence Intervals

A crucial difference lies in interpreting uncertainty:

  • Confidence Interval (Frequentist): ‚ÄúIf we repeat the experiment infinitely, 95% of the calculated intervals will contain the true value of \(\theta\).‚Äù

    ‚ö†Ô∏è \(\theta\) is fixed ‚Äî it‚Äôs either in the interval or not.

  • Credible Interval (Bayesian): ‚ÄúThere is a 95% probability that \(\theta\) lies within the interval.‚Äù

    üîë Conditional on the prior being correct.
